# Roadmap

## Phase 1 — Playable Core

Goal: create a stable browser-playable foundation for account progression, city ownership, and core economy.

| Feature | Outcome |
| --- | --- |
| Login/Register | Players can create accounts and return securely. |
| City scene | Phaser city scene displays the player city. |
| Land buying | Players can purchase available plots. |
| Building construction | Players can place buildings and wait for completion. |
| Coin/Diamond | Core currency balances are visible and server-backed. |
| Backend API | REST endpoints support player and game state. |
| Basic shop | Players can browse and buy initial building items. |

## Phase 2 — Content, Polish, and Retention

Goal: make the city feel alive and give players more reasons to return.

| Feature | Outcome |
| --- | --- |
| 30+ buildings | Expanded progression and variety. |
| Animated cars | Roads and city loops feel active. |
| Trees | Visual polish and decoration. |
| Weather | Rain, snow, and atmosphere effects. |
| Day/Night | Time-of-day lighting and ambience. |
| Sound | UI feedback, city ambience, construction audio. |
| Daily missions | Repeatable goals for retention and rewards. |

## Phase 3 — Multiplayer and Advanced Economy

Goal: evolve Real Estate Empire into a social MMO strategy economy.

| Feature | Outcome |
| --- | --- |
| Multiplayer | Shared systems connect players beyond individual cities. |
| Chat | Global, clan, and trade communication. |
| Bank | Deposits, timed products, loans, or investment features. |
| Stock market | Simulated market for late-game strategy. |
| Auction | Competitive bidding for rare assets. |
| Trading | Player-to-player item or asset exchange. |
| Clans | Group identity, missions, and shared competition. |
| Global events | Time-limited server-wide challenges. |

## Development Priorities

1. Keep Phase 1 small and shippable.
2. Prefer stable backend-authoritative game state over visual polish when a tradeoff is required.
3. Add content only after the core loop is reliable.
4. Add multiplayer only after economy exploits and account state are under control.
